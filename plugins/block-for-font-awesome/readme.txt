Display a Font Awesome 5 icon in a Gutenberg block.

== Description ==

This plugin allows you to display any Font Awesome 5 icon as an editor block (Gutenberg).

You can also display inline icons by using the `[fa class="fas fa-fw fa-phone"]` shortcode.

Read more about the [Block for Font Awesome plugin](https://getbutterfly.com/wordpress-plugins/block-for-font-awesome/) here.

== Installation ==

1. Log into your WordPress site
2. Install and activate plugin
3. Add a Font Awesome Icon block
